# Rock Paper Scissors Game
A console-based Rock Paper Scissors game developed using Java. The game allows users to play multiple rounds against the computer, validates user input, tracks scores, and displays the final match result.
## Features
- User-defined number of rounds
- Input validation for Rock, Paper, and Scissors
- Random computer move generation
- Round-by-round winner display
- Score tracking for user, computer, and draws
- Final match result declaration.
## Technologies Used
Java
- Scanner Class
- Random Class
## Concepts
- Loops (for, while)
- Conditional Statements (if-else)
- Switch Statement
- String Handling
- Input Validation
- Random Number Generation
- Score Tracking
